<Tab title="Creating Endpoints">
### Creating an Endpoint
    <Steps>
        <Step>
            <span style={{display:'flex', alignItems: 'flex-start', justifyContent: 'space-between', marginTop: '-10px', padding:'0 10px 0 10px'}}>
                <span>
                    <img style={{width: '3.5em', borderRadius: '5px', background: '#7E7CBE', padding: '1px'}}
                src="/images/link.svg"
                alt="link icon" nozoom />
                </span>
                <span style={{marginLeft: '15px'}}>
                    To start creating an endpoint, begin by navigating to the Endpoints section on the left Sidebar indicated by the link icon.  
                </span>
            </span>
        </Step>

        <Step>
            <span style={{display:'flex', alignItems: 'center',padding:'0 10px 0 10px'}}>
                <span>
                    <img src="/images/endpoints-add.png"
                alt="add endpoint screenshot" />
                </span>
                <span style={{marginLeft: '15px'}}>
                    Click the 'plus' button to create a new endpoint.
                </span>
            </span>
        </Step>

        <Step>
            Give your new Endpoint a name, an optional description, whether it is to be enabled or disabled (this can also be updated later), and select the Canopy if there is a Canopy you wish to add the endpoint to.
        </Step>

        <Step>
            Call your new Endpoint via a GET request or in your browser by navigating to the generated URL.
        </Step>

    </Steps>
